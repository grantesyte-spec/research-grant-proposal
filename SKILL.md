### Step 2: Aggregate Results

**After ALL research files complete:**



### Step 3: Generate Proposal

**Using the unified research_notes.md:**

Generate the proposal citing ONLY verified real references.

**Output Structure:**
1. Background & Significance
2. Objectives & Content
3. Methods & Technical Approach
4. Expected Outcomes & Innovation
5. References (from all sources)

## GOLDEN RULES

1. **Use the RIGHT file** for the RIGHT database
2. **Each verification_steps file is self-contained** - it generates its own output
3. **SKILL.md coordinates** - doesn't do research itself
4. **Research FIRST** - then aggregate → generate
5. **Record everything** in the appropriate result file
