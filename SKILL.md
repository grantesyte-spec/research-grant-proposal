# Research Grant Proposal Generator

**IMPORTANT**: This skill follows a modular RESEARCH-FIRST workflow.

## Core Principles

1. **Research First** - Never generate proposals without real literature
2. **Verify Everything** - All references must be validated
3. **Modular Research** - Each database has its own workflow
4. **Coordinate via SKILL.md** - This file orchestrates the entire process

---

## Golden Rules

1. **Use the RIGHT file** for the RIGHT database
2. **Each verification_steps file is self-contained** - it generates its own output
3. **SKILL.md coordinates** - doesn't do research itself
4. **Research FIRST** - then aggregate → generate
5. **Record everything** in the appropriate result file
6. **Verify ALL references** before citing in proposal

## Citation Format

**In-Text Citations:**
`[1]`, `[2]`, `[1][2]`, `[1]-[3]`

**Reference List:**
```
[number] Authors. Title[J]. Journal Name, Year, Volume(Issue): Pages. DOI. Verification URL: https://...
```

**Example:**
```
[1] Wang Y, Zhang X, Liu J. Effects of nursing intervention on hip fracture[J].
    J Clin Nurs, 2022, 31(15): 2156-2165. DOI: 10.1111/jocn.16000.
    Verification URL: https://pubmed.ncbi.nlm.nih.gov/35012345/
```
